# CCIDE для CC:Tweaked

`ccide.lua` — автономный внутриигровой редактор Lua. Дополнительные библиотеки и
доступ к интернету ему не нужны.

## Установка

1. Открой Advanced Computer в Minecraft.
2. Перенеси `ccide.lua` в компьютер через стандартную загрузку файлов CC:Tweaked.
   Проще всего перетащить файл мышью в открытый терминал компьютера, находясь в
   обычной оболочке CraftOS.
3. При желании переименуй файл в `ccide`.
4. Запускай редактор вместе с нужным файлом:

```text
ccide plan.lua
```

Если имя файла не передано, IDE откроет новый буфер. `Ctrl+O` позволяет открыть
файл позднее.

## Управление

- колёсико мыши — прокрутка по три строки;
- левый клик — установка курсора;
- клик или перетаскивание по правой полосе — быстрая прокрутка;
- `Ctrl+S` — сохранить;
- `Ctrl+O` / `Ctrl+N` — открыть / создать файл;
- `Ctrl+Z` / `Ctrl+Y` — отмена / повтор;
- `Ctrl+F`, затем `F3` — поиск и следующее совпадение;
- `Ctrl+G` — перейти к строке;
- `F5` — сохранить и запустить программу, затем вернуться в IDE;
- `Ctrl+Q` — выход с предложением сохранить изменения;
- `Tab` / `Shift+Tab` — отступ / уменьшение отступа;
- `F1` — встроенная справка.

Полноценные события мыши доступны на Advanced Computer. На обычном компьютере
редактором можно пользоваться с клавиатуры.

## Ограничения первой версии

- одновременно редактируется один файл;
- подсветка рассчитана на обычный Lua-код и не разбирает многострочные строки;
- выделение текста мышью и системный буфер обмена для копирования пока не
  реализованы; вставка через событие `paste` работает.
