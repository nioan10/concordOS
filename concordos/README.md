# ConcordOS — базовая промышленная ОС

Это первый запускаемый слой для компьютера CC:Tweaked: русское отображение, рабочий стол, безопасная загрузка и запуск существующих программ.

## Установка

1. На клиенте Minecraft включи ресурс-пак из папки `resourcepack`. Его нужно упаковать в zip так, чтобы `pack.mcmeta` лежал в корне архива, затем поместить в `.minecraft/resourcepacks` и включить в настройках. После этого перезайди в мир или нажми `F3 + T`.
2. Скопируй в компьютер только один файл `install_concordos.lua` и выполни `install_concordos`. Он развернёт ОС и сохранит прошлый `/startup` как `/startup.before_concordos`.
3. Скопируй нужные приложения (`ccide.lua`, `plan.lua`, `checklist.lua`, `inspect_create.lua`) в корень компьютера.
4. Выполни `reboot`.

## Обновления

Начиная с ConcordOS 0.3.0 обновления устанавливаются командой `update` или плиткой «Обновления» в разделе «Инструменты». Обновлятор читает манифест из GitHub, подкачивает только системные файлы и не затрагивает `/concordos/data` с заявками. Для работы на сервере должна быть разрешена HTTP API CC:Tweaked. После обновления выполни `reboot`.

## Что уже есть

- Русский терминал с UTF-8 вводом/вставкой и CP866-выводом; для Lua-программ используй `run /файл.lua`.
- Русские UTF-8 строки в программах и CP866-вывод на терминал через `system/lib/ru.lua`.
- Рабочий стол с мышью, колесом, клавиатурой и автоматическим скрытием отсутствующих приложений. На главном пульте остаются «Мастер промзоны», терминал и редактор; «План производства», чеклист и инспектор собраны внутри плитки «Инструменты».
- Загрузчик с маркером незавершённого запуска, режимом восстановления и журналом `/concordos/logs/crash.log`.
- «Мастер промзоны»: разовые и постоянные заказы Create. Постоянная заявка хранится в `/concordos/data/orders.db`; фоновая служба проверяет её раз в 15 секунд, а повторные запросы остатка ограничивает экспоненциальной паузой от 30 секунд до 2 минут.
- Вкладка «Каталог» в «Мастере промзоны» показывает позиции Stock Ticker с остатками, поиском и страницами. Клик по позиции переносит её в форму постоянной заявки.
- В «Мастере промзоны» кнопка «Из блокнота» читает недостающие материалы из подключённого планшета Create (`create:clipboard`). Выбор позиции переносит её ID и количество в форму постоянной заявки; остаётся указать адрес доставки и подтвердить заявку.

## Важно

Функция `ru.encode()` нужна только перед выводом на экран. Внутри логики, конфигов, имён квакопортов и рецептов используй обычный UTF-8: `"Склад латунных листов"`. Сравнения без учёта регистра делай через `ru.equalsIgnoreCase(a, b)`.

Пока CCIDE остаётся самостоятельной программой и выводит обычный UTF-8. Его русификация (включая курсор и редактирование по символам, а не байтам) — следующий отдельный шаг, чтобы не сломать уже работающий редактор.
